# Space Trash!

A game similar to Asteroids, made with vanilla HTML+JS+CSS using HTML Canvas.

Play it on Github Pages: https://afbcary.github.io/space-trash/

## How to Play
* You are the yellow dot
* Control yourself with the arrow keys
* Don't get hit by the trash floating around
* New trash spawns from the edges of the screen
* Surviving longer results in a higher score
